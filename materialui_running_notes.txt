About Material UI:
React components for faster and easier web development.
It gives a variety of options for making your app stand out


Installation:
Run yarn add @material-ui/core


Icons:
Installing dependency: yarn add @material-ui/icons
Importing:
import IconName from '@material-ui/icons/Icon'
Adding them as an attribute:
1. startIcon = { <IconName /> } -- for adding icon in the start
2. endIcon = { <IconName /> } -- for adding icon in the end


Components -- ComponentName (Attributes):
1. Button
    (variant="contained/outlined",
    color="primary/secondary",
    size="small/large)
    Note: Make it a href, a plain button or add keyword "disabled" to disable it
2. FormControlLabel -- Making forms in React
    (control = {pass any thing which goes in forms})
3. Checkbox (checked, inputProps, 
    icon={<IconName />}
    checkedIcon={<IconName />}
    )
4. TextField -- Adding Inputs
    (variant="filled/outlined", color="primary/secondary",
    type="date/time/email", placeholder="whatever you wanna write")

Changing the Entire Theme:
import { ThemeProvider, createMuiTheme } from '@material-ui/core/styles'
import { color1, color2 } from '@material-ui/core/colors' 
const theme = createMuiTheme({
    palette: {
        primary: {
            main: color1[val],
        },
        secondary; {
            main: color2[val]
        },
    }
})
We can nest themes (for inner and outer themes) [Read Documentation]
